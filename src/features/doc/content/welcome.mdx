## Stack

- Next.js 16
- Tailwind CSS v4
- shadcn/ui

## Featured

- Clean, modern layout with light and dark themes
- vCard integration
- SEO ([JSON-LD](https://json-ld.org), sitemap, robots)
- AI-friendly [/llms.txt](https://llmstxt.org)
- Spam-protected email
- Installable as a PWA
- Analytics with [PostHog](https://posthog.com) and [OpenPanel](https://openpanel.dev)
- Consent management via [c15t](https://c15t.com)

## Content

Centralized MDX:

- One content layer for blog posts and component docs
- Category-based organization
- Raw `.mdx` endpoints for tooling
- Syntax-highlighted code blocks
- Dynamic OG images for link previews
- RSS

## Registry

Reusable components, hooks, and blocks distributed with the [shadcn CLI](https://ui.shadcn.com/docs/cli), with live previews and install commands.
